# Market Brief Agent 🤖📊

An autonomous AI research agent that delivers a daily market briefing covering 26 global instruments across US, European, and Asian equities, ETFs, and macro signals.

## What It Does

Every morning at 07:00 CET, the agent:
- Gathers live market data, news, and macro signals
- Reasons through opportunities and risks
- Produces a structured brief with clear, actionable calls
- Publishes the brief to a public dashboard

## Action Vocabulary

| Call | Meaning |
|---|---|
| **BUY** | Open or add to a position |
| **ACCUMULATE** | Buy gradually on weakness |
| **HOLD** | Do nothing; thesis intact |
| **TRIM** | Sell part, keep the rest |
| **SELL** | Close the position |
| **AVOID** | Don't enter now |
| **WATCH** | Monitor for a trigger |
